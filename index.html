Developers can now control digital lighting with a friendly, extensible Javascript API. This API allows developers to create unprecedented immersive interactive experiences.

Search for a compatible device. If you are on a Mac, after plugging in your device run sudo kextunload -b com.FTDI.driver.FTDIUSBSerialDriver in Terminal.

Use Cases

Protect Valuable Artwork

Using room occupancy sensors, dim the gallery lighting when the space is unoccupied to protect valuable artwork from excessive light exposure.

Create New Media Exhibits

Creative lighting has a long history in New Media Art, but was overly technical and complicated. With this API you’re free to be completely creative.

Add Color to Your Desktop

Add colorful accent lighting around your workstation or home entertainment system. Sync the lights to your music for your own mini-concert.

Transform a Space

Control the lighting of a space based on the time of day or any number of other variable… or allow visitors to take control.

Liberating Content From the Screen

Traditionally all the content in interactive multi-touch installations lives within the boundaries of the screen (other than audio). Controllable lighting allows us to match the environment to the graphics that users are interacting with onscreen. As easy as it is to play a sound effect, developers can now effortlessly tap into this entirely new dimension of creative potential.

The API Layers

Experience Layer

The experience layer is comprised of the methods and properties that end-developers will use to build their applications. Using a defined vocabulary of words, such as .color() or .intensity() developers can control all the functions of digital lighting devices regardless of how the data is actually transferred to the lighting fixtures.
